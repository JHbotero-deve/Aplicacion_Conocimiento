# 🐄 Proyecto de Gestión Ganadera

Sistema de gestión ganadera con **Node.js + Express + PostgreSQL** en el backend y un **frontend estático** para interacción básica.  
Incluye autenticación con JWT, pruebas de API (`requests.http`) y scripts de auditoría.
